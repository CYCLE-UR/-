## 📲 創建二維碼

### 線上工具

使用這些工具生成二維碼：

1. **QR Code Generator**
   - https://www.qr-code-generator.com/
   - 輸入網址：`https://cycle-ur.github.io/-/user-system-v2.html`
   - 下載二維碼圖片

2. **QRCode Monkey**
   - https://www.qrcode-monkey.com/
   - 可自定義顏色和 Logo

3. **命令行生成**
   ```bash
   # 使用 qrencode
   echo "https://cycle-ur.github.io/-/user-system-v2.html" | qrencode -o qrcode.png
   ```

### 二維碼用途

- 📱 貼在海報上
- 💬 發送到群組
- 📧 放在郵件簽名
- 🖼️ 社交媒體帖子

## 🔧 進階分享

### 自定義域名

1. **購買域名**（如：ticketbot.com）
2. **在 GitHub Pages 設置自定義域名**
3. **更新 DNS 記錄**

```
# 添加 CNAME 記錄
www  CNAME  cycle-ur.github.io
```

### 短網址

使用服務創建短網址：

- bit.ly：https://bit.ly/ticketbot
- tinyurl.com：https://tinyurl.com/ticketbot
- is.gd：https://is.gd/ticketbot
