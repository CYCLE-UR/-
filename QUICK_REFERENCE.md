# 快速參考卡 - 智能搶票助手

## 🎯 30 秒快速開始

### 前台搶票（用戶）
```
1. 進入 http://127.0.0.1:8080/demo.html
2. 點擊「🚀 開始使用」
3. 點擊「➕ 新增任務」
4. 填寫：名稱、時間、網址、點擊選擇器
5. 點擊「▶️ 開始」
```

### 後台管理（管理員）
```
1. 進入 http://127.0.0.1:8080/admin.html
2. 輸入：admin / admin123
3. 查看儀表板和監控面板
4. 管理所有用戶數據
```

## 🎮 前台功能菜單

### 導航欄
- 👤 顯示當前用戶統計
- 🚪 登出按鈕

### 主要功能
1. **任務列表** - 卡片式展示所有任務
2. **新增任務** - ➕ 按鈕打開表單
3. **統計分析** - 查看成功率和趨勢
4. **設置** - 通知權限管理

### 任務操作
- ▶️ **開始** - 立即執行
- ⏸️ **暫停** - 暫停執行
- ▶️ **繼續** - 恢復執行
- 🗑️ **刪除** - 移除任務

---

## 🔐 後台功能菜單

### 登入屏幕
```
帳號：admin
密碼：admin123
```

### 側邊欄導航
1. **📊 儀表板** - 系統概覽
2. **👥 用戶管理** - 用戶列表
3. **📋 任務監控** - 所有任務
4. **📈 統計分析** - 圖表報表
5. **🖥️ 系統監控** - 性能監控
6. **🔔 通知記錄** - 通知歷史
7. **⚙️ 系統設置** - 配置和危險操作

### 儀表板統計卡片
- 👥 總用戶數 → 活躍用戶數
- 📋 總任務數 → 進行中任務
- ✅ 成功率 → 成功任務數
- 📅 今日任務 → 昨日比較
- ⏳ 待執行 → 已暫停數
- ❌ 失敗任務 → 需要關注
- 💾 存儲使用 → 已用 MB
- ⚡ 平均執行 → 秒/任務
